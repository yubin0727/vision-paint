# vision-paint
My computer vision painter using OpenCV

- **Free Drawing Painter**
  - Handling mouse events
  - We can draw point, line, ellipse, rectangle
  - 'Tab' : change brush color
  - '+' : increase the radius
  - '-' : decrease the radius
  - (space) : change brush mode
